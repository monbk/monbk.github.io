两尾猫

发表于 2017-09-08 | 分类于 小说 | 热度: ℃

两尾猫有两个尾巴，
两尾猫住在山上，
两尾猫跟着人跑啦，
两个尾巴掉光啦。

暑假的时候回乡下，我在老家的后山上遇见了一只两尾猫。

第一年的暑假，是我自己偷跑上了山。

它的前爪被兽夹困住，很可怜的样子，安静的等我打开。我想应该是从没见过人的。它把爪子伸出来，开始还一瘸一拐的，走了几步路伤口就愈合了，身轻如燕，我也是这个时候才发现原来它有两条尾巴。他走了几步又停下来，端坐在我面前，瞳孔眯成了一道缝。我想，这八成是已成了精。

它眨了眨眼，变成了一个少年。『多谢相助。』

我想，这确实是已成了精。

两尾猫变成的少年告诉我，它们一族是一起住在太行山上的，不过成年前可以自己出来玩，长大了再回去。它来到这儿第一次见到兽夹，好奇碰了一下，就动也动不了，躺了三天，遇到了我。我笑了，一半因为这种奇遇，一半因为这个可爱的少年。

暑假剩下的日子，我和少年腻在了一起，每天攀爬嬉戏，好不自在。我走的时候，他站在山头送我，依依不舍。

第二年的暑假，我见到了熊。

放下书包后，我迫不及待的跑向去年走时和少年约定的地点，那里却传来了一种陌生的、恐怖的吼声。是一只熊，棕色的大熊。迅猛有力的扑了过来。在我吓呆的时候，两尾猫出现了，和他斗在一起。一切结束之后，熊和猫一起躺在地上死了。

如果我没有上山……我抱起了两尾猫小小的尸体，嚎啕大哭，哭到太阳落山，哭到鼻涕流进了嘴里。我擦擦眼睛，休息了一下，发现两尾猫掉了一条尾巴。尾巴落在地上就不见了，两尾猫小小的身体开始回暖。

两尾猫复活了。

它身上恐怖的伤口也开始缓慢的愈合，虚弱的摇了摇尾巴，又变成了少年。少年抬头看着哭成了傻逼的我，笑了。

『你怎么哭得这么难看。抱歉忘记告诉你，我有两条命啦。』

原来两尾猫的两条尾巴就是它的两条命。我又哭又笑，怪他不早点告诉我，他笑嘻嘻的说，

『我救了你……你能不能亲亲我？』

原来他和我想的一样，我终于忍不住笑了出来。

两尾猫伤好的很快，表明心意后，我把它抱回了家，反正就剩下一条尾巴，就当在山上捡来的大猫。暑假结束的时候，它告诉我他要走了。

『还有一个月我就成年啦，要回家啦。』『再等等，等暑假结束吧。』『不行不行，等不了了。』等不及去见可爱的小公猫们吧，我酸溜溜的想着，把山猫扔出了家门，山猫熟练的从窗子里翻了进来。
『我舍不得你，可是我得走了。』它用鼻子拱我，用湿漉漉的舌头舔我的脸，我才意识到，它是猫，不是人。

我点了点头，它走得无声无息。

第三年的暑假，我不死心的回到了后山，早已猫去山空。

妖怪就是无情，我躺在床上，怨恨的想到。舅爷看我闷闷不乐，要给我讲故事，他是以前是山上的猎人，没打过几次熊，却藏了一肚子的山野故事，我就问他，『舅爷，你见过两尾猫吗？』

他一下子变了脸色，『你从哪里知道的两尾猫？记得离这东西远点。』

我不明白，他接着讲，『两尾猫之所以有两条尾巴，是因为从前有只猫得罪了庙里的老道士，道士就下了一道符，把它的尾巴分成了两条，命呢，也有了两条。这可不是什么好事情，因为每只两尾猫成年的时候都要受到符咒的惩罚，痛苦的死去，掉下一条尾巴丢掉一条命，作为赎罪。』
